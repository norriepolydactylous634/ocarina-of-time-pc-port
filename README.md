## 📌 Project Overview

This software allows you to play the original Ocarina of Time on your PC. It provides a native experience without the need for an emulator. It runs directly on your Windows hardware. You get higher frame rates, widescreen views, and better textures than the original game console.

## 💻 System Requirements

Your computer needs to meet these basic standards to run the game well:

- Operating System: Windows 10 or Windows 11 (64-bit).
- Processor: Intel Core i5 or AMD Ryzen 5 processor from the last five years.
- Memory: 8 GB of RAM.
- Graphics: A dedicated graphics card with support for DirectX 11.
- Storage: 500 MB of free space for game files.
- Controller: A standard USB game controller. While you can use a keyboard, a controller provides a better experience.

## ⚙️ Installation and Setup

Once you download the files, follow these steps to prepare the game.

1. Locate the file you downloaded in your Downloads folder.
2. Right-click the file and select Extract All. 
3. Choose a folder on your computer where you want to keep the game files. Do not use your Program Files folder.
4. Open the folder you just created.
5. You must provide a copy of your own Ocarina of Time ROM file. This is a requirement for the build process to function. Rename this file to `rom.z64`.
6. Move the `rom.z64` file into the game folder.
7. Double-click the file named `Shipwright.exe` or `Launcher.exe` to start the game.

- Resolution: Select the resolution that matches your screen.

- Controller Mapping: Select your controller from the list. Follow the on-screen prompts to map buttons to your controller inputs.

## 🛠 Troubleshooting Common Issues

If the game does not start, check these items.

- Missing ROM: The game requires the original file to function. Ensure you placed the `rom.z64` file in the main folder.
- Graphics Drivers: Ensure your computer graphics drivers are up to date. Visit the manufacturer website for your graphics card to get the newest version.
- Antivirus software: Sometimes security software blocks new programs. If you receive a warning, check your security software to allow the game to run.
- Controller connection: Plug your controller into a different USB port if the computer does not recognize it.
